# Overdue Radar

Minimal GitHub Pages site for a single-client Xero custom integration.

## Files

- `index.html`: public landing page
- `privacy.html`: basic privacy policy

## Planned use

This repository is intended to host:

- a simple public-facing project page for Xero app registration
- a privacy policy URL

## GitHub Pages

After the repository is pushed to GitHub:

1. Open repository `Settings`
2. Open `Pages`
3. Set `Source` to `Deploy from a branch`
4. Select branch `main` and folder `/ (root)`

The published site URL will look like:

`https://<username>.github.io/overdue-radar/`
